# Sam — Developer Portfolio

Personal portfolio site for [@ifeelsam](https://github.com/ifeelsam), built with Next.js, TypeScript, and Tailwind CSS.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

Edit content in `src/lib/site.ts` — bio, projects, skills, and social links.

Add your email in `site.ts` and wire it in `src/components/contact.tsx` if you want a mailto link.

## Deploy

Deploy to [Vercel](https://vercel.com) or any Next.js host:

```bash
npm run build
```
